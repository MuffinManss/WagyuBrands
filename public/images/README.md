# Required Images

Place the following image files in this folder for the site to display correctly:

| File | Usage |
|------|-------|
| `og-cover.png` | Open Graph / social sharing (1200×630 recommended) |
| `pofuyu-char.png` | Character art (Moonmaru) |
| `mango-sunflowers.gif` | Character art (Macarune) |
| `mango-sushi.gif` | Gallery / character scene |
| `mango-void.gif` | Gallery / character scene |
| `character-scene.jpeg` | Story section, gallery |

Without these assets, images will appear broken. Add your actual image files here before deploying.
